En este paso estamos entrenando un rbol de decisin para regresin usando todas las variables disponibles, con configuracin predeterminada. Tras entrenarlo, evaluamos su desempeo utilizando:

Error cuadrtico medio (MSE): Indica la diferencia cuadrtica promedio entre las predicciones y valores reales. Raz del error cuadrtico medio (RMSE): Facilita la interpretacin en las mismas unidades que la variable objetivo. Coeficiente de determinacin (R): Mide qu tan bien se ajustan las predicciones al valor real.

Ajuste de profundida del rbol con valores variados

Se ajusta la profundidad mxima del rbol de decisin, probando varios valores (3, 5, 7, 10 y 15). Se evala el desempeo mediante las mtricas (MSE, RMSE, y R). Este anlisis te permitir identificar cul es la profundidad ptima para tu modelo, equilibrando la precisin con la capacidad generalizadora del modelo.

In[10]:
depths = [3, 5, 7, 10, 15]
results = []

for depth in depths:
    tree = DecisionTreeRegressor(max_depth=depth, random_state=42)
    tree.fit(X_reg_train, y_reg_train)
    pred = tree.predict(X_reg_test)
    mse = mean_squared_error(y_reg_test, pred)
    rmse = np.sqrt(mse)
    r2 = r2_score(y_reg_test, pred)
    results.append((depth, mse, rmse, r2))

results_df = pd.DataFrame(results, columns=['Max Depth', 'MSE', 'RMSE', 'R'])
display(results_df)

plt.figure(figsize=(8, 5))
sns.lineplot(x='Max Depth', y='RMSE', data=results_df, marker='o')
plt.title('RMSE segn profundidad del rbol de regresin')

plt.ylabel('RMSE')
